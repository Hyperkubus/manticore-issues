# Project Manticore — Issue Tracker

Bug reports and feedback for **Project Manticore**, a stealth intelligence thriller where you command a covert reconnaissance vessel.

## Reporting

[**Open a new issue**](https://github.com/Hyperkubus/manticore-issues/issues/new/choose) and pick a template:

- **Bug Report** — something broke, glitched, or behaved wrong
- **Feedback** — confusion, things that felt off, observations about how it plays
- **Feature Request / Idea** — something you wanted to do but couldn't

All feedback is welcome, including feature requests — the earlier we hear them, the easier to plan for. Don't worry about perfect formatting; a rough description beats no report.

## Manual

The tester walkthrough lives at **[hyperkubus.github.io/manticore](https://hyperkubus.github.io/manticore/)** — controls, mechanics, mission types, and what to look for.
